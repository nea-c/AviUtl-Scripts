| -: | :- |

| 中心Y | 中心位置Yを操作します。(サイズ外処理未対応) |

| 幅 | 画面サイズYから割合で幅を指定します。 |

| ダイアログ | 内容 |
| -: | :- |
| 帯色 | 色を指定します。 |
| pxモード | 幅設定を割合からpxに変更します。 |
