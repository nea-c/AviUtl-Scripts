# Scatter

個別オブジェクトを拡散させるスクリプト。

- [patch.aul](https://github.com/ePi5131/patch.aul)必須

## 最新 / Latest

**v1.2**

## Scatter

| トラックバー | 内容 |
| -: | :- |
| progress | 進行度(%)を操作します。 |
| scale | 倍率(%)を操作します。 |

| ダイアログ | 内容 |
| -: | :- |
| x | Xの拡散範囲を指定します。 |
| y | Yの拡散範囲を指定します。 |
| z | Zの拡散範囲を指定します。 |
| rx | rxの回転範囲を指定します。 |
| ry | ryの回転範囲を指定します。 |
| rz | rzの回転範囲を指定します。 |
| zoom | 拡大範囲を指定します。 |
| seed | 乱数のシードを指定します。 |
