# 画面アスペクト

画面アスペクトを指定してサイズ確認等をできるようにするスクリプト

## 最新 / Latest

**r3**

## 各パラメータ


* ### X比率 / Y比率
> 画面比率を指定します。

* ### 適応度
> どれくらい適応するかを指定します。
> 0%にするとシーン解像度になります。

* ### 色
> 色を指定します。

* ### 反転
> シネマスコープのような挙動に変化します。

* ### ピクセル数表示
> 現在表示している画面比率のピクセル数を表示します。

* ### 文字サイズ
> ピクセル数表示のチェックがONの時に表示される文字のサイズを指定します。
