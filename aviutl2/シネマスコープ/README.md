## 各パラメータ

* ### 中心X / 中心Y
> シネマスコープの中心位置を変更します。

* ### 角度
> 中心位置を基準に回転します。

* ### オフセット
> 中心位置と角度を基準にして移動します。

* ### 幅
> どれくらい見えるかを指定します。

* ### 色
> 色を指定します。
